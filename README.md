#■STI 單表繼承 (Single Table Inheritance) - 蘇介吾 105/08/20

##安裝：
1. git clone https://github.com/afgnsu/sti.git
2. rvm install 2.0.0
3. rvm use 2.0.0
4. rvm gemset create rails400
5. rvm use 2.0.0@rails400
6. gem i rails -v 4.0.0
7. cd sti
8. rm -rf Gemfile.lock
9. bundle
10. rake db:migrate
12. rails s
